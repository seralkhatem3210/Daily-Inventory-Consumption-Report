Periodically Consumed Report for Odoo 18

Analyze how products are consumed over any date range. See total consumed, average daily, on-hand now, and coverage days—filtered by one or more warehouses.

Key Features

  • Generate period reports (weekly, monthly, custom ranges)
  • Filter by specific warehouses or include all internal locations
  • Metrics: Total Consumed, Avg Daily, On-Hand Now, Coverage (days)
  • List, Pivot and Graph views with search, filters and group-bys
  • Counts any move leaving the selected warehouse set (smart internal→internal handling)
